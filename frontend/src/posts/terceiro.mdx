### Código de exemplo

Aqui está um exemplo de código React:

```jsx
import { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Você clicou {count} vezes</p>
      <button onClick={() => setCount(count + 1)}>Clique aqui</button>
    </div>
  );
}
```

> MDX é uma excelente maneira de combinar conteúdo de texto com componentes interativos React.
